# PHP chat server

A PHP chat server written in [Ratchet](http://socketo.me/) using [React PHP](https://reactphp.org/) for non binding I/O.

## Install

1. Clone this repo.
2. If you're using [ahoy](ahoy.com) then run
`ahoy up`

otherwise:

`docker-compose up -d`

3. To view logs run

`ahoy logs` or `docker-compose logs`
